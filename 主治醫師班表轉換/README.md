# 麻醉部主治醫師班表轉換程式

將「麻醉部值班工作表」主治醫師班表，自動轉換為「詳細班表上傳表」格式。

---

## 使用環境

Google Colab（或任何支援 Python 3 + pandas 的環境）

---

## 使用步驟

### 1. 上傳檔案

在 Colab 左側檔案欄上傳以下三個檔案（上傳至 `/content/`）：

- `202604麻醉部值班工作表.csv`
- `attending_convert_schedule.py`

### 2. 執行程式

在 Colab 儲存格執行：

```python
exec(open('attending_convert_schedule.py').read())
```

### 3. 下載結果

執行完成後，左側檔案欄會出現 `詳細班表上傳表_轉換結果.csv`，右鍵點擊下載。

---

## 編碼問題排除

若開啟 CSV 出現亂碼，或執行時讀檔報錯，請將程式中的 `utf-8-sig` 與 `big5` 互換後重試。

---

## 輸出符號說明

| 符號 | 意義 |
|------|------|
| `D1` | 台北白班 |
| `D2` | 淡水白班 |
| `DN1` | 台北白班＋值班 |
| `DN2` | 淡水白班＋值班 |
| `N1` | 台北純值班（無白班） |
| `T` | 值班後隔天（接班） |
| `O` | 門診 |
| `P` | 疼痛業務# |
| `v` | 外院支援* |
| `%` | ICU 醫師（全月） |
| `V` | 一般上班日（其餘未標註） |

若同一天有兩種符號（如前一天值班的 `T` 加上當天門診 `O`），程式會以逗號合併（如 `T,O`）並在執行結果中提示，請人工確認。

---

## 注意事項

- 本程式僅處理**主治醫師**（大寫英文代號），住院醫師（小寫代號）會略過。
- 台北／淡水分界以表格中「淡水」欄位為準，新增醫師代號不影響判斷邏輯。
- 每月更換新的工作表時，只需替換輸入檔案名稱（程式第一行的路徑），其餘邏輯不需修改。
