# 門診表轉換程式

從「麻醉部值班工作表」自動產生「麻醉評估門診表」。

---

## 使用環境

Google Colab（或任何支援 Python 3 + pandas 的環境）

---

## 使用步驟

### 1. 上傳 CSV

在 Colab 左側檔案欄，將當月的值班工作表 CSV 上傳至 `/content/`。

### 2. 貼上程式碼

在 Colab 新增一個儲存格，將程式碼完整複製貼上。

### 3. 換月時修改三處

貼上後找到程式碼開頭，修改年份、月份與檔名：

```python
YEAR  = 2026
MONTH = 4

df = pd.read_csv('202604麻醉部值班工作表.csv', ...)
```

### 4. 填入白班R（週六及其他需要補排的日子）

原始值班工作表的白班R欄為空白，執行程式前請先手動填入：

- **台北白班R**：台北醫師欄位區塊最右側的「白班R」欄，週六若無門診V 或門診R，填入當天上白班的住院醫師代號
- **淡水白班R**：淡水醫師欄位區塊最右側的「白班R」欄，淡水週六通常無門診，填入當天上白班的住院醫師代號

填好後存檔，再執行程式，白班R 即會自動帶入門診表。

### 5. 執行並下載結果

執行儲存格後，左側檔案欄會出現 `門診表_轉換結果.csv`，右鍵點擊下載。

---

## 來源欄位與優先順序

台北與淡水各自依以下順序取門診人員，取到第一個有值的欄位為準：

| 優先順序 | 欄位 | 說明 |
|---------|------|------|
| 1 | 門診V | 主治醫師門診 |
| 2 | 門診R | 住院醫師門診 |
| 3 | 白班R | 排班者事後填入，週六白班住院醫師兼任門診 |

---

## 編碼問題排除

若開啟 CSV 出現亂碼，或執行時讀檔報錯，請將程式碼中的 `utf-8-sig` 與 `big5` 互換後重試。
