# 門診表轉換程式

從「麻醉部值班工作表」自動產生「麻醉評估門診表」。

---

## 使用環境

Google Colab（或任何支援 Python 3 + pandas 的環境）

---

## 使用步驟

### 1. 上傳 CSV

在 Colab 左側檔案欄，將當月的值班工作表 CSV 上傳至 `/content/`。

### 2. 貼上程式碼

在 Colab 新增一個儲存格，將程式碼完整複製貼上。

### 3. 換月時修改三處

貼上後找到程式碼開頭，修改年份、月份與檔名：

```python
YEAR  = 2026
MONTH = 4

df = pd.read_csv('202604麻醉部值班工作表.csv', ...)
```

### 4. 執行並下載結果

執行儲存格後，左側檔案欄會出現 `門診表_轉換結果.csv`，右鍵點擊下載。

---

## 來源欄位說明

台北與淡水各自依以下優先順序取門診人員，取到第一個有值的欄位為準：

1. 門診V
2. 門診R
3. 白班R（取第一個代號）

---

## 編碼問題排除

若開啟 CSV 出現亂碼，或執行時讀檔報錯，請將程式碼中的 `utf-8-sig` 與 `big5` 互換後重試。
