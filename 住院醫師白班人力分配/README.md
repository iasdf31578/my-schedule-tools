# 每日人力分配排班程式

根據麻醉科值班工作表，自動排出每日台北白班、淡水白班、及 OFF 名單。

---

## 使用環境

Google Colab 或任何 Python 3.10+ 環境，需安裝 `pandas`。

---

## 每月使用步驟

### 1. 準備輸入檔案

將當月的排班工作表從 Google Sheet 下載為 CSV，確認欄位名稱如下：

| 欄位名稱 | 說明 |
|---------|------|
| `Unnamed: 1` | 星期（數字 1–7 或中文一–日皆可） |
| `台北值班` | 當日台北值班醫師代號 |
| `淡水值班` | 當日淡水值班醫師代號 |
| `台北門診` | 當日台北門診醫師代號 |
| `淡水門診` | 當日淡水門診醫師代號 |
| `假` | 當日請假醫師代號（單人如 `k`、多人可逗號分隔如 `k,m` 或直接連寫如 `km`） |

### 2. 修改程式頂部的設定區

打開 `scheduler_v2.py`，只需修改以下三個地方：

```python
# ① 本月人員身分（每月更新）
DOCTOR_TYPE = {
    "k": "A",   # A - 刀房內，主要院區台北
    "n": "A",
    "m": "B",   # B - 刀房內，主要院區淡水
    "r": "B",
    "t": "B",
    "j": "C",   # C - 特殊科，主要院區台北
    "o": "C",
    "s": "C",
    "u": "C",
    # D - 特殊科，主要院區淡水（無人則不填）
}

# ② 上個月最後一天的值班人員
PREV_TAIPEI_ONCALL  = "k"   # 台北值班，多人用逗號分隔，例如 "k,n"
PREV_DANSHUI_ONCALL = "t"   # 淡水值班

# ③ 檔案路徑
INPUT_FILE  = "202605.csv"
OUTPUT_FILE = "202605_output.csv"
```

### 3. 執行程式

在 Colab 直接執行整個 cell，完成後會輸出 `OUTPUT_FILE` 指定的 CSV 檔案，並在畫面顯示預覽。

## 排班邏輯

| 步驟 | 說明 |
|------|------|
| ① OFF | 前一日台北值班 ＋ 前一日淡水值班 |
| ② 排除名單 | OFF ＋ 今日值班 ＋ 台北／淡水門診 ＋ 請假 |
| ③ 值班者入主體 | 台北值班身分 A/B/D → 台北主體；身分 C → 台北括號 |
| | 淡水值班身分 A/B/C → 淡水主體；身分 D → 淡水括號 |
| ④ A/B 交替補入 | 台北優先選 A，淡水優先選 B，交替分配直到 A/B 用完 |
| ⑤ 補平 | 若台北主體人數 < 淡水，從剛分配給淡水的 A/B 中移一人（優先 A）至台北 |
| ⑥ 括號外無人時 | 若台北或淡水主體為空，將該院區值班者從括號移出至主體 |
| ⑦ 剩餘 C/D | 所有剩餘 C → 台北括號；所有剩餘 D → 淡水括號 |

**輸出格式**：主體人員直接連接，括號內人員加括號，例如 `knr(josu)`

**星期六**：只輸出 OFF（來自星期五值班），不排白班。

---

## 警告訊息

若星期五出現以下情況，程式會印出警告，需手工確認調整：

> ⚠️  警告 X（五）台北／淡水白班：特殊科 [代號] 因人力不足被移出括號，請手工確認調整！

觸發條件（三者同時成立）：
1. 當天是**星期五**
2. 該院區括號外**主體為空**
3. 被移出括號的值班者身分為 **C 或 D**（特殊科）

## 人員身分說明

| 身分 | 說明 |
|------|------|
| A | 刀房內人員，主要院區台北 |
| B | 刀房內人員，主要院區淡水 |
| C | 特殊科人員，主要院區台北 |
| D | 特殊科人員，主要院區淡水 |

---

## 編碼設定

程式預設使用 `utf-8-sig`（Google Sheet 下載的 CSV 通常為此格式）。
若出現編碼錯誤，請將設定區的 `ENCODING` 改為 `"big5"`。
