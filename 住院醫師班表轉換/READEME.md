# 住院醫師班表轉換程式

從「住院醫師值班表（手動確認版）」自動產生「詳細班表上傳表」格式。

---

## 使用環境

Google Colab（或任何支援 Python 3 + pandas 的環境）

---

## 使用步驟

### 1. 上傳 CSV

在 Colab 左側檔案欄上傳以下兩個檔案至 `/content/`：

- `202604麻醉部值班工作表.csv`
- `202604住院醫師值班表-手動確認.csv`

### 2. 貼上程式碼

在 Colab 新增一個儲存格，將程式碼完整複製貼上。

### 3. 執行前手動填入三項設定

貼上後找到程式碼開頭，依當月情況填入：

```python
# 本月國定假日日期（數字），例如清明連假：{3, 6}
NATIONAL_HOLIDAYS = {3, 6}

# 守房間期間（R1）的住院醫師代號，週六白班標 DS 而非 D1/D2
# 例如前三個月的住院醫師：{'j', 'k'}
R1_CODES = set()

# 外訓其他地區（不回來值班）的住院醫師代號
# 例如：{'m'}
ET_CODES = set()
```

### 4. 換月時修改兩處檔名

```python
df_in   = pd.read_csv('202604住院醫師值班表-手動確認.csv', ...)
df_main = pd.read_csv('202604麻醉部值班工作表.csv', ...)
```

### 5. 執行並下載結果

執行後左側檔案欄會出現 `住院醫師詳細班表_轉換結果.csv`，右鍵點擊下載。

---

## 轉換邏輯

### 優先順序

**S > DO > LS > V**，其餘依班別類型判斷。

### 符號對照表

| 條件 | 輸出符號 |
|------|---------|
| 國定假日（使用者指定） | `S` |
| 週日 | `OFF` |
| OFF 欄（前日值班） | `DO` |
| 週六空白 / 週六請假 | `LS` |
| 平日請假 | `V` |
| 台北值班 | `DN11` |
| 淡水值班 | `DN22` |
| 平日門診（台北或淡水） | `O1` |
| 週六門診 | `OS` |
| 平日白班（台北或淡水） | `D` |
| 週六台北白班 | `D1` |
| 週六淡水白班 | `D2` |
| 週六白班（R1 守房間期間） | `DS` |
| 外訓其他地區 | `ET` |

---

## 警告說明

執行結果末尾若出現 ⚠️，代表以下情況需人工確認：

- 當天不在任何欄位中（非週六日），程式輸出 `?`
- 同一天同時出現多個非預期的班別欄位組合

---

## 編碼問題排除

若開啟 CSV 出現亂碼，或執行時讀檔報錯，請將程式碼中的 `utf-8-sig` 與 `big5` 互換後重試。
