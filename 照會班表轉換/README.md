# 照會班表轉換程式

從「麻醉部值班工作表」自動產生「照會班表」。

---

## 使用環境

Google Colab（或任何支援 Python 3 + pandas 的環境）

---

## 使用步驟

### 1. 上傳 CSV

在 Colab 左側檔案欄，將當月的值班工作表 CSV 上傳至 `/content/`。

### 2. 貼上程式碼

在 Colab 新增一個儲存格，將程式碼完整複製貼上。

### 3. 換月時修改檔名

貼上後找到程式碼中的這一行，將檔名改為當月的檔案：

```python
df = pd.read_csv('202604麻醉部值班工作表.csv', header=None, encoding='big5')
```

### 4. 執行並下載結果

執行儲存格後，左側檔案欄會出現 `照會班表_轉換結果.csv`，右鍵點擊下載。

---

## 轉換邏輯

### 平日（有住院醫師值班R的日子）

| 欄位 | 來源 |
|------|------|
| 白天會診V | 當天二線V |
| 白天會診R | 當天值班R |
| 夜間會診V | 當天值班V |
| 夜間會診R | 當天值班R |

### 假日（六、日，以及無住院醫師值班R的國定假日）

| 欄位 | 來源 |
|------|------|
| 白天會診V | 當天值班V |
| 夜間會診V | 當天值班V |
| R欄 | 留空 |

> **注意**：程式以「當天有無住院醫師值班R」判斷平假日。  
> 本科國定假日目前均無住院醫師值班，因此國定假日會自動走假日路徑，無需額外設定。  
> 若未來國定假日開始排住院醫師值班，請將該日加入程式頂端的 `NATIONAL_HOLIDAYS`，例如：
> ```python
> NATIONAL_HOLIDAYS = {3}  # 4/3 為補假
> ```

---

## 編碼問題排除

若開啟 CSV 出現亂碼，或執行時讀檔報錯，請將程式碼中的 `utf-8-sig` 與 `big5` 互換後重試。

---

## 輸出格式

共 8 行，台北、淡水各 4 行：

| 院區 | 班別 | 人員 |
|------|------|------|
| 台北 | 日 | 主治 |
| 台北 | 日 | 住院 |
| 台北 | 夜 | 主治 |
| 台北 | 夜 | 住院 |
| 淡水 | 日 | 主治 |
| 淡水 | 日 | 住院 |
| 淡水 | 夜 | 主治 |
| 淡水 | 夜 | 住院 |
