# 麻醉部排班系統

## 流程說明

1. **取得麻醉部值班工作表** - 從google sheet下載excell檔，轉成csv檔，此時encoding= big5
2. **確認請假住院醫師** - 將"麻醉部值班工作表"中的值班R,門診R欄位複製貼上至"住院醫師值班工作表"，並且填入請假名單- 人工
3. **排出住院醫師白班人力** - 安排平日白班住院醫師-自動
4. **排入周六人力** - 於"住院醫師值班表"以及"麻醉部值班工作表"中，填入周六白班R與門診R- 人工

**排入周六人力後**

6. **住院醫師班表轉換** - 根據"住院醫師值班表"以及"麻醉部值班工作表"(為了取得住院醫師編碼)轉換成上傳表-自動
7. **主治醫師班表轉換** - 根據"麻醉部值班工作表"轉換成上傳表-自動
8. **生成照會表** - 根據"麻醉部值班工作表"產出照會排班表-自動
9. **生成門診表** - 根據"麻醉部值班工作表"產出門診排班表-自動
